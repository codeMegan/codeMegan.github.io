“从心学医学”第8期：心腔-右心室

右心室简介

  1. 右心室直接位于胸骨左缘第4、5肋软骨的后方，右心房的前下方，心内注射多在胸骨旁左侧第4肋间隙注入右心室。
  2. 右心室壁厚0.3-0.4cm，仅是左心室壁厚的1/3，而且供应血管相对较少。
  3. 右心室以室上嵴为界分为流入道（窦部）和流出道（漏斗部）两部分。
  4. 室上嵴是位于右房室口与肺动脉口之间的一弓形肌性隆起。

右心室流入道

  1. 从右房室口延伸至右心室尖部。
  2. 室壁有多条纵横交错的肌性隆起，称肉柱。
  3. 流入道室壁上有突入室腔的锥状肌隆起，称乳头肌。
  4. 各乳头肌的尖端移行为数条腱质的细索，即腱索。
  5. 根据乳头肌附着于右心室壁位置的不同，可分为前乳头肌、后乳头肌、隔侧乳头肌三组。
  6. 前乳头肌15个，位于右心室前壁中下部，其尖端发出510条腱索，呈放射状散开，连于三尖瓣前、后尖；后乳头肌较小，有2~3个，位于下壁，乳头肌尖端发出的腱索多数连于三尖瓣后尖；隔侧乳头肌（圆锥乳头肌）最小，但数目最多，发出的腱索多连于隔侧尖。
  7. 右心室内有一起自室间隔连于右室前壁乳头肌根部的肌束，称隔缘肉柱，又称节制索。其内有心传导系纤维（右束支）通过。隔缘肉柱有防止右心室过度扩张的作用。

右房室口

  1. 右心室流入道的入口为右房室口，呈卵圆形。
  2. 右房室口的周缘有由致密结缔组织构成的右房室口纤维环，环上附有三个呈三角形的瓣膜，称三尖瓣。
  3. 三尖瓣按部位可分为前尖、后尖和隔侧尖。
  4. 三尖瓣环、三尖瓣、腱索和乳头肌在结构上和功能上是一个整体，称三尖瓣复合体，其作用是防止血液逆流。

右心室流出道

  1. 右心室流出道，又称动脉圆锥或漏斗部，位于右心室前上部，壁光滑，呈锥体状，上端为肺动脉口，通入肺动脉干腔。
  2. 肺动脉口周缘有三个彼此相连的瓣环形纤维环，称肺动脉瓣环，环上附有三个袋状的肺动脉瓣，分别排列在肺动脉口的左、右、前方，开口向肺动脉干腔。
  3. 肺动脉瓣环游离缘中央有一小结，称半月瓣小结。
  4. 当心室收缩时，血液冲开肺动脉瓣，流入肺动脉干；心室舒张时，三个肺动脉瓣彼此互相靠拢，使肺动脉口封闭，阻止血液逆流回右心室。
